# 2025-02-05 Memory

## A股行情数据获取

### 问题发现
- stock-fetcher.py 脚本包含大量模拟/示例数据（"示例股票A/B"、随机数等）
- AKShare API 连接不稳定（`RemoteDisconnected` 错误）
- 指数和个股数据经常获取失败，只有板块数据较稳定

### 解决方案
- 创建了 stock-fetcher-fixed.py（纯净版），移除所有模拟数据
- 只在获取到真实数据时才输出，否则显示"暂无数据"
- 板块数据可以正常获取，指数和个股数据经常失败

### 当前状态
- 板块数据：✅ 正常
- 指数数据：❌ 连接失败
- 个股数据：❌ 连接失败
