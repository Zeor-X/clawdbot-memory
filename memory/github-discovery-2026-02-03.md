## 🛠️ 发现的工具和辅助项目

### 3. **CowAgent**
- 📍 GitHub: cowagent/cowagent
- 🎯 基于大模型的超级 AI 助理
- ✨ 特性：
  - 主动思考和任务规划
  - 访问操作系统和外部资源
  - 创造和执行 Skills
  - 长期记忆和成长
  - 支持飞书、钉钉、企业微信、微信公众号、网页
  - 多模型：OpenAI/Claude/Gemini/DeepSeek/Qwen/GLM/Kimi/LinkAI
  - 处理文本、语音、图片和文件

### 4. **1Panel - Linux 服务器管理面板**
- 📍 1Panel-dev/1Panel
- 🎯 直观的 Web 界面管理 Linux 服务器
- ✨ 特性：
  - 管理 OpenClaw agents
  - LLMs 管理
  - 网站、数据库管理
  - 容器、文件、计划任务管理
  - Go 语言开发

### 5. **Grok Companion (Waifu Container)**
- 📍 热门项目，自托管 Grok Companion
- 🎯 一个"灵魂容器"，用于创造虚拟伙伴
- ✨ 特性：
  - 实时语音聊天
  - 支持 Minecraft、Factorio 游戏玩
  - 跨平台：Web / macOS / Windows

### 6. **Claude Code Skill (Manus-style)**
- 🎯 Claude Code 技能，实现 Manus 风格的持久化 Markdown 规划
- 💰 背后是 $2B 收购的工作流模式
- 🔧 适用：持续规划和工作流管理

### 7. **Agent Skills for Obsidian**
- 🎯 Obsidian 的 Agent 技能
- 📝 用于在 Obsidian 中集成 AI Agent 功能

### 8. **Memory for 24/7 Proactive Agents**
- 📍 Python 项目
- 🎯 为 24/7 主动代理提供记忆功能
- 🎯 适用 OpenClaw (moltbot, clawdbot)

### 9. **Antigravity Model Proxy**
- 🎯 代理工具，暴露 Antigravity 提供的 Claude/Gemini 模型
- 🎯 目的：在 Claude Code 和 OpenClaw 中使用

### 10. **ClawdBot 一键部署工具**
- 🎯 Shell 脚本工具
- 🎯 快速部署 Clawdbot

### 11. **Crabwalk 🦀**
- 🎯 TypeScript 项目
- 🎯 OpenClaw Agents 的实时伴生监控器

### 12. **OpenClaw 汉化版**
- 📍 JavaScript/TypeScript
- 🎯 OpenClaw 中文版
- ✨ 特性：
  - 全中文 CLI + Dashboard
  - 全流程搭建教程和排错指南
  - Claude/ChatGPT LLM 接入
  - WhatsApp/Telegram/Discord 多平台
  - 每小时自动同步

### 13. **Token Usage Tracker**
- 📍 TypeScript CLI 工具
- 🎯 跟踪多个工具的 token 使用情况
- 📊 支持：
  - OpenCode
  - Claude Code
  - OpenClaw (Clawdbot/Moltbot)
  - Codex
  - Gemini CLI
  - Cursor IDE
  - AmpCode
  - Factory Droid
- 🏆 全球排行榜 + 2D/3D 贡献图
